## Installation Guide
### Prerequisities
System dependencies:
```
$ node -v
  v10.15.3

$ npm -v
  6.4.1

$ webpack -v
  4.34.0
```

#### Local setup
##### Clone project
```
$ git clone https://github.com/btuerker/js-weather-app.git
$ cd ./js-weather-app
```
##### Install project dependencies
```
$ npm install
```
##### Build project
```
npm run build
```

or 

```
$ yarn install
$ yarn build
```

Now you can check the ./dist/index.html file with your best browser.
